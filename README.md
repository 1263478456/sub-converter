# SubConverter 订阅转换

基于 [SubConverter-Extended](https://github.com/Aethersailor/SubConverter-Extended) + [SubWeb](https://github.com/stilleshan/subweb) 的订阅转换服务。

## 部署

```bash
docker compose up -d
```

- 前端：`http://IP:3215`
- 后端：`http://IP:3216`

配置文件：`/mnt/vol3/appdata/subconverter/base/pref.toml`
